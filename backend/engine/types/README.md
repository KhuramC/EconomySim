# Types

Types are consolidated within this folder so that they can be easily found from outside the backend.

The types provided are:

- [Demographic](./demographic.py): The demographics associated with `PersonAgent`s.
- [DemoMetrics](./demographic_metrics.py): The demographic metrics supported by the simulation.
- [IndustryType](./industry_type.py): The type associated with each `IndustryAgent`.
- [Indicators](./indicators.py): The economic indicators supported by the simulation.
- [IndustryMetrics](./industry_metrics.py): The industry metrics supported by the simulation.
- [PricingType](./pricing_type.py): The pricing strategies supported by the simulation.
